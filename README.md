# Algorand Voting DApp

## 📌 Project Description
This project is a decentralized voting application built on the Algorand blockchain. It allows users to cast votes securely and transparently.

## 🚀 Features
- Secure voting using blockchain
- Transparent vote counting
- Tamper-proof records

## 🛠️ Tech Stack
- Algorand Blockchain
- Python / JavaScript (SDK)
- Smart Contracts (ASC1 / PyTeal)

## 📂 Project Structure
- /contracts → Smart contracts
- /frontend → User interface
- /scripts → Deployment scripts

## 🎯 Goal
To build a secure and decentralized voting system using Algorand.
